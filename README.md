# YouTube Thumbnail Downloader

A fast, free, and responsive YouTube thumbnail downloader tool that allows you to download high-quality thumbnails from any YouTube video instantly.

## Features

- **Multiple Resolutions**: Download thumbnails in SD, HQ, HD, Full HD, and 4K quality
- **100% Free**: No registration or payment required
- **Fast & Secure**: Instant thumbnail extraction with no data storage
- **Mobile Friendly**: Works perfectly on all devices
- **SEO Optimized**: Built with best practices for search engine visibility

## Technologies Used

- Vue.js 3 - Progressive JavaScript Framework
- Vite - Next Generation Frontend Tooling
- CSS3 - Modern styling with gradients and animations

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## How to Use

1. Copy the YouTube video URL
2. Paste it into the input field
3. Click "Get Thumbnail"
4. Choose your preferred resolution and download
